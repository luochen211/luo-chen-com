# 每日单词小程序

原生微信小程序 MVP，包含首页、单词学习、单词挑战和周报四个页面。学习数据采用本地缓存和微信云数据库双层持久化。

## 运行

1. 打开微信开发者工具。
2. 选择“导入项目”，目录指向本文件夹。
3. 未配置 AppID 时可直接使用游客模式预览。此时数据会保存在当前设备，但不会跨设备同步。

## 启用云端永久保存

1. 在 `project.config.json` 中将 `touristappid` 替换为正式 AppID，然后重新导入项目。
2. 在微信开发者工具的“云开发”面板中创建一个云环境。
3. 在云数据库中创建 `user_progress` 集合，权限设为“仅创建者可读写”。小程序不直接读写该集合，所有操作由云函数按 `openid` 隔离。
4. 在开发者工具中右键 `cloudfunctions/syncProgress`，选择“上传并部署：云端安装依赖”。
5. 清除小程序缓存后重新进入，学习一个单词；在云数据库的 `user_progress` 集合中确认已生成当前用户记录。

如果云端暂时不可用，页面仍会立即写入 `wx.setStorageSync`；下次启动或下次修改数据时会再次同步。同步冲突时保留 `updatedAt` 更新的整份状态。

## 持久化数据

- 每日学习目标与按日学习数量
- 已掌握单词、待复习单词和收藏
- 最近 1000 条学习记录与答题记录
- 由真实记录计算的连续学习天数、七日图表和正确率
